Variables and Definitions
------------------------------------------

Total Sales by Total Returns: The comparison of each sub category with sales vs returns. The sub categories a color coded with each circle on the scatterplot. Hovering over the circles will give you more information on sub category, total sales, and total returns

Return Rate Per Category: The return rate of each category(technology, office supplies, furniture) shown in vertical bar visuals

Return Rate Per Customer: Filtered out all customers that did not have any returns, Only showing the average return rate of customers that made returns. Bar chart sorted in descending order to average return rates.

Return Rate Per State: Measures the avg rate of each return in each state. Higher pigmented tones of red indicate higher returns. Hovering over each state shows the return rate

Return Rate Per Quarter: Shows the return rate per quarter.

Return Rate by Month shows the trend between the months and avg return rate. As you can see there is a peak of returns around July and August. There is an upward trend on the returns as the year progresses

Return Rate by Month + Category shows the average  return rate  per month of the year for each category. 

Return Rate by Month + Sub Category shows the average  return rate  per month of the year for each sub category. Just more detail into possible peaks around the end of the year

AVG Return Rate + Total Sales Per state gives more insight on sales made vs returns made with each state per category(color coded).
